# Security Policy

## Supported versions

Security fixes are provided for the latest published version of OverflowBar.

## Reporting a vulnerability

Please use **Security → Report a vulnerability** in the GitHub repository to submit a private security advisory. Include the affected macOS version, OverflowBar version, reproduction steps, and the expected impact. Do not include screenshots or logs containing private menu bar information unless they are necessary and have been redacted.

Please avoid opening a public issue until the report has been reviewed.
