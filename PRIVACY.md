# Privacy

OverflowBar runs locally on your Mac and does not include analytics, advertising, telemetry, or network data collection.

## Permissions

- **Accessibility** is used to discover menu bar controls, invoke supported press actions, and perform user-requested menu bar arrangement changes.
- **Screen Recording** is used to capture the small visual regions belonging to selected menu bar items so they can be represented in the overflow row.

Captured menu bar images are held in memory for the current app session. OverflowBar does not upload them or write them to a remote service.

## Local data

OverflowBar stores selected item identifiers, layout preferences, onboarding completion, hover behavior, and login-item state locally using macOS system preferences. Removing the app does not automatically remove those preferences.

## Reporting concerns

For a potential security or privacy issue, use the repository's private GitHub Security Advisory reporting flow rather than posting sensitive details in a public issue.
