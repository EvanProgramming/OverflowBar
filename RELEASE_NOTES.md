# OverflowBar 1.0.0

The first public build of OverflowBar, a native menu bar overflow manager for Apple Silicon Macs running macOS 15 or later.

## Highlights

- Compact second-row overflow panel with adaptive Liquid Glass on macOS 26.
- Choose which accessible menu bar items are managed by OverflowBar.
- Hybrid activation for clicking mirrored menu bar controls.
- Notch, safe-area, narrow-screen, full-screen, and multiple-display positioning.
- Guided first-run setup for Accessibility and Screen Recording permissions.
- First-run customization for managed icons, hidden layout, and launch at login.
- Reduced Motion support and subtle system-style interactions.

## Install

1. Download `OverflowBar-1.0.0.dmg`.
2. Open the disk image and drag OverflowBar to Applications.
3. Launch OverflowBar and follow the four-step welcome setup.

## Compatibility

- Apple Silicon
- macOS 15 or later
- Liquid Glass appearance on macOS 26 or later

## Signing notice

This community build is ad-hoc signed and is not yet Apple-notarized because the project does not currently have a Developer ID certificate. macOS may require Control-clicking the app and choosing **Open** on first launch. The SHA-256 checksum is included as a separate release asset.
